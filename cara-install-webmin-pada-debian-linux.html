Cara install Webmin pada Debian Linux

Apa itu Webmin? Kurang lebih Webmin adalah Web Administration, yaitu sebuah aplikasi dengan antarmuka berbasis web untuk administrasi sistem berbasis Linux/Unix. Dengannya anda dapat mengatur account pengguna, setup Apache, DNS, file sharing, dan masih banyak lagi. Jadi selain anda dapat mengunakan remote access secara cli seperti telnet dan ssh, anda juga dapat mengakses system Linux anda melalui web dengan tampilan Gui yang menawan tentunya..

Kurang lebih ada dua cara yang biasa dilakukan untuk menginstall webmin pada system operasi Linux versi Debian, termasuk RaspbianOS untuk Raspberry Pi board tentunya. Pilihan pertama dapt menggunakan file instalasi package untuk debian, yaitu file berekstensi .deb, dan pilihan kedua menggunakan apt repository, yang mana saya lebih merekomendasikan cara kedua ini, karena bisa dengan mudah melakukan update hanya dengan apt-update command. Mari kita lihat satu-persatu cara yang telah saya jelaskan di atas.

Pilihan pertama: menggunakan debian package

  • Download debian package dari terminal dengan perintah berikut
    sudo wget http://prdownloads.sourceforge.net/webadmin/webmin_1.890_all.deb
  • jalankan perintah berikut pada terminal untuk menginstall webmin
    sudo dpkg --install webmin_1.890_all.deb
  • Jika terdapat depedency package yang hilang, coba instal pacakge berikut:
    sudo apt-get install perl libnet-ssleay-perl openssl libauthen-pam-perl libpam-runtime libio-pty-perl apt-show-versions python

Cukup singkat cara pertama tersebut bukan? Sekarang kita coba cara yang ke dua kedua, yaitu installasi lewat APT Repository

  • Pertama-tama login as root
  • Edit file "/etc/apt/sources.list" dan tambahkan baris berikut
    deb https://download.webmin.com/download/repository sarge contrib, atau anda dapat gunakan perintah berikut langsung pada terminal
    echo "deb https://download.webmin.com/download/repository sarge contrib " | sudo tee -a /etc/apt/sources.list
  • Install GPG Key, ketikkan perintah beikut pada terminal ...ingat satu-persatu
    cd /root

    wget http://www.webmin.com/jcameron-key.asc

    apt-key add jcameron-key.asc
  • Kemudian jalankan perintah berikut satu-persatu juga ya
    apt-get install apt-transport-https

    apt-get update

    apt-get install webmin
  • Jika terjadi error, coba tambahkan perintah berikut
    apt –fix-broken install
  • Jika telah selesai, anda dapat mengelola Webmin melalui alamat https://<ip-address-anda>:10000 pada browser anda, akan tampil tampilan login seperti berikut, gunakan login yang biasa anda pergunakan untuk masuk ke dalam system linux anda.
  • Berikut tampilan dashboard dari webmin itu sendiri setelah anda berhasil login.
  • Jika port anda ter-block oleh firewall, anda dapat meng-edit iptables dengan cara berikut
    Edit file "/var/lib/iptables" dan tambahkan baris berikut
    -A INPUT -p tcp -m tcp --dport 10000 -j ACCEPT
    Jika telah selesai, restart iptables dengan perintah berikut
    /etc/init.d/iptables restart

Demikianlah cara instalasi webmin pada system operasi Linux versi Debian dan variantnya.
