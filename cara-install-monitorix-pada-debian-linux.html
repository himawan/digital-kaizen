Cara install Monitorix pada Debian Linux

Monitorix adalah aplikasi open source yang lumayan ringan dan  dipergunakan sebagai alat monitoring sistem, dirancang untuk memantau layanan dan sumber daya sistem sebanyak mungkin. Aplikasi monitorix ini banyak dipergunakan pada server produksi Linux / UNIX, namun karena kesederhanaan dan ukuran kecil juga dapat digunakan juga untuk memantau perangkat perangkat embedded.

Aplikasi ini memperlihatkan kepada anda sebuah live chart pada dashboard utamanya sehingga jika terjadi sesuai yang janggal pada grafiknya, anda dapat segera berjaga-jaga atau melakukan analisa lebih jauh untuk mengetahui problem yang sebenarnya terjadi tentunya, 

Jika kalian ingin mencoba aplikasi ini untuk memonitoring server atau perangkat IT anda agar dapat mengetahui perkembangannya, yang sangat kami sarankan ini, silahkan ikuti langkah langkah berikut. 

  • Tambahkan Repository key dengan perintah berikut pada terminal satu-persatu
    echo "deb http://apt.izzysoft.de/ubuntu generic universe" | sudo tee -a /etc/apt/sources.list

    wget http://apt.izzysoft.de/izzysoft.asc

    sudo apt-key add izzysoft.asc
  • Lakukan apt-update kemudian install package monitorix
    sudo apt-get update
    sudo apt-get install monitorix -y
  • Jalankan service monitorix
    service monitorix start
  • Untuk mengakses monitorix, buka browser anda dan masuk ke alamat berikut http://your-ip-address:8080/monitorix

  • Ketika service monitorix dimulai, ia akan membaca file konfigurasi yang berada pada “/etc/monitorix/monitorix.conf”, yang dapat anda set sesuai dengan option yang anda inginkan, seperti menghidupkan dan mematikan grafik.

    Contohnya untuk men-set heading pada home page, gunakan variable berikut:
    title = title-anda
    Untuk mengganti nama hostname pada grafik, gunakan variable berikut:
    hostname = nama-hostname-anda
    Untuk dapat meng-customize lebih jauh lagi, anda dapat melihat dokumen resmi dari monitorix itu sendiri ya.
  • Yang perlu diperhatikan oleh para pengguna Debian, Monitorix juga punya file konfigurasi tambahan yang berada pada /etc/monitorix/conf.d/00-debian.conf yang berisikan opsi-opsi khusus untuk system debian.File konfigurasi ini akan di-load setelah konfigurasi utama, dimana opsi yang terdapat pada file konfigurasi utama akan di overwrite oleh file konfigurasi tambahan ini.
  • Jika anda telah selesai melakukan perubahan pada file konfigurasi, silahkan restart service dari monitorix itu sendi dengan cara
    service monitorix restart
    atau
    systemctl restart monitorix

Cukup sekian cara instalasi Monitorix pada system operasi Linux versi Debian, yang dapat anda gunakan juga pada RaspbianOS pada Raspberry Pi. Selamat memantau....
